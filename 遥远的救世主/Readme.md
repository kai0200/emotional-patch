# 遥远的救世主

- 道和规律的入门

## 第一章 分析肖雅文

- 要在有能力的人身边，做长远的眼光，有自己的底线。
- 会办事要有形式，求人办事的态度是很重要的，了解人性（找芮小丹这条线保持了和丁元英的关系）。
- 肖雅文也是刑警对出来的。
- 芮小丹在第一章也在用形象（丁元英抽烟喝酒泡妞）来判断一个人，主观判断并不准确。没有肖雅文有智慧。
- 敢于投资花了大价钱来维护关系。



## 评价丁元英

是魔是鬼就不是人，为什么？首先要知道什么是人，贪婪、自私是人、高尚慈悲也是人。但丁元英不被这些标签所定义。既不是好人、也不是坏人。
有思想不对、没有思想也不对。丁元英窥见了入道的门槛。
