## 庄子

《逍遥游》的核心内容就是告诉你，真正的逍遥自在不应该依附于食物、权力、名声、甚至身体。
“无待”，不依附于任何东西的快乐，才是真的逍遥。

《齐物论》齐生死，同万物。
形其槁木，心其死灰，才能同万物而齐生死，不论是非、好坏，齐而接受之。
人籁、地籁、天籁同而视之，道过窍，自鸣之？非也，道是之然也。
“夫吹万不同，而使其自己也，咸其自取，怒者其谁邪？”
庄子讲道使之然也，齐万物而与道合一也。无论情绪、生死都是道法自然也，你理解了就不会纠结于
自己的情绪变化，不惧怕生死。

## 人有悲欢离合，月有阴晴圆缺 - 苏轼

抱残守缺 -  看透人生的的智慧，你不可能什么都完美，有所缺才是生活。

经典依据：
《道德经》中多次强调“守柔”“处下”“知止”。例如：

“大成若缺，其用不弊”（最完美的事物看似有缺陷，但其作用永不衰竭）；

“为学日益，为道日损”（求道需放下执念，回归朴素）；

“曲则全，枉则直”（委曲反能保全，弯曲反能伸直）。

核心逻辑：
道家认为自然本身充满“不圆满”的状态（如月亮圆缺、四季轮转），强求完美违背天道。“抱残守缺”实质是顺应自然、接受无常的智慧，通过放下执念达到心灵自由。


## 淮南子

夫喜怒者，道之邪也；忧悲者，德之失也；好憎者，心之过也；嗜欲者，性之累也。

人大怒破阴，大喜坠阳；薄气发喑，惊怖为狂；忧悲多恚，病乃成积；好憎繁多，祸乃相随。

故心不忧乐，德之至也；通而不变，静之至也；嗜欲不载，虚之至也；无所好憎，平之至也；不与物散，粹之至也。

能此五者，则通于神明。 - 《淮南子》

## 道德经

“至虚极，守静笃。
万物并作，吾以观复。
夫物芸芸，各复归其根。
归根曰静，是谓复命。
复命曰常，知常曰明。
不知常，妄作凶。
知常容，容乃公，公乃王，王乃天，天乃道，道乃久，没身不殆。”

通过内心极致的虚静，来体认宇宙万物循环往复、回归本根的永恒规律；认识并顺应这一规律，人才能获得真正的智慧，实现生命的安顿与长久。

“至虚极，守静笃”不仅是这一章的起点，也是老子整个哲学体系中一个非常重要的修行法门。
