# Read book

## method 1

1. 整理书中段落知识点和逻辑结构线条
2. (可见与不可见)
观察知识体系整体，有那些写了，有哪些没写？
3. (空白与沉默)
从这些部分，能分析推理出一个时代的背景，作者思维和知识储备，等多种因素。
4. (共鸣与生产)
生产性阅读，不是单纯依附站在巨人的肩膀上看世界，也不是刁钻的批判者，而是把书中作者原有的思维和自己的思维并列结合起来，探索生产出新思路，加以补齐。

## method 理解

1. 尽信书不如无书，所以第一点是总结精髓，大部分的理论都大差不差，王阳明讲无心自足，很多只是大部分人已经理解，l对己有用的实际不多。

2. 其次要去思考，可能并不是要动笔，总结思路，理解作者在写书的思路、逻辑，是否这个逻辑会让其丢失什么，比如《国富论》的漏洞就衍生出了《资本论》。
而《资本论》 的问题只研究了生产关系、财富剥削。并没有引出信仰、情感、社会认识等相关问题。

3. 要思考和作者共同进步，观点都有其局限性，你所在的时空对同一观点也会有不同的认识，有些观点已经不适合当前的时代和你自己的身心情况。学着与作者一起进步。

## 持续读书的方法

1. 笔记选择名词、定义，这里的逻辑就是一定要让脑子去总结简单的知识。
2. 每次读完一段以后，仔细考虑他写着一段的逻辑即可。
3. 人类的大脑喜欢总结类似的知识，不要把问题搞复杂。
